All Help and Instructions

To get a c code converted to js, first upload the c file using the "upload" button and then use the "convert" button to convert to JavaScript.

You can also write your code directly to editor provided, and then use the button "convert".

Use the button "Execute" to execute the js code. The results will be shown in the console of browser.

C2JS deals with only C99 version of the C language. Apart from this version restriction, there are also some other constraints due to run time environment of browser and operating system(s). A detailed discussion about restrictions and limits can be found in sections 2.1, 2.3, 2.4 and 2.5 of final documentation.

About Section

A "C to JavaScript compiler", that efficiently translates a "standard C" source program to equivalent JavaScript (as well as necessary HTML) and thus can be executed in a web browser or an equivalent emulator. C2JS will first validate the original C code, run it through all the traditional compiler front-end phases (Lexical Analysis, Syntax Analysis, Semantic Analysis), and then translate the source file into valid equivalent JavaScript.

C2JS is developed as a graduation project for the degree requirements of Computer Science under the faculty of information technology, University of Central Punjab Lahore, Pakistan.
